# Provenance

`bioma-npc` started from the upstream `npc-engine` repository:

<https://github.com/ethz-gtc/npc-engine>

This repository retains inherited code and research materials from that project,
including the lumberjacks scenario and multiple planner examples.

To preserve attribution and upstream notice requirements:

* original contributor credits remain listed in [AUTHORS.txt](AUTHORS.txt)
* the preserved upstream MIT notice remains in [LICENSE-MIT](LICENSE-MIT)
* README citation and acknowledgment sections continue to reference the original
  research context where appropriate

BiomaAI is extending this codebase for agentic MMO and multi-agent simulation
systems. New BiomaAI-authored additions and repository-level distribution terms
are described in [LICENSE](LICENSE).
